# Material Yazi Flavor

This directory provides the Material Deep Ocean flavor for Yazi.

- Deep ocean blues with vivid accents, based on material-deep-ocean
- Background: #0F111A, Foreground: #A6ACCD
- Primary accent: #82AAFF, Secondary: #89DDFF, Success: #C3E88D, Warning: #FFCB6B, Error: #FF5370

Use it by setting in your ~/.config/yazi/theme.toml:

```toml
# Your overrides ...

theme = "material"
```

Yazi will merge your theme with this flavor automatically.
