# Tokyodark Yazi Flavor

This directory provides the Tokyodark flavor for Yazi.

- Dark, high-contrast blue-accented variant inspired by tokyodark-dark
- Background: #11121d, Foreground: #a0a8cd
- Primary accent: #7aa2f7, Secondary: #7dcfff, Highlights: #bb9af7, Success: #9ece6a, Warning: #e0af68, Error: #f7768e

Use it by setting in your ~/.config/yazi/theme.toml:

```toml
# Your overrides ...

theme = "tokyodark"
```

Yazi will merge your theme with this flavor automatically.
