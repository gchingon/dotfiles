# Nebulous Yazi Flavor

This directory provides the Nebulous Twilight flavor for Yazi.

- Twilight-inspired cool palette based on nebulous-twilight
- Background: #1b1f2a, Foreground: #c5c8d1
- Primary accent: #84a0c6, Secondary: #89b8c2, Success: #b4be82, Warning: #e2a478, Error: #e27878

Use it by setting in your ~/.config/yazi/theme.toml:

```toml
# Your overrides ...

theme = "nebulous"
```

Yazi will merge your theme with this flavor automatically.
