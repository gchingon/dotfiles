# Eldritch Yazi Flavor

This directory provides the Eldritch Darker flavor for Yazi.

- Moody violet/teal on deep midnight, based on eldritch-darker
- Background: #171928, Foreground: #d8e6e6
- Primary accent: #8b75d9, Secondary: #0396b3, Success: #2dcc82, Warning: #ccd663, Error: #cc5860

Use it by setting in your ~/.config/yazi/theme.toml:

```toml
# Your overrides ...

theme = "eldritch"
```

Yazi will merge your theme with this flavor automatically.
