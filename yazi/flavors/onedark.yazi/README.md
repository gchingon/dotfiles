# Onedark Yazi Flavor

This directory provides the Onedark flavor for Yazi.

- Deep One Dark variant with cyan/blue accents, based on onedark-deep
- Background: #0b0e14, Foreground: #a0a8b7
- Primary accent: #61afef, Secondary: #56b6c2, Success: #98c379, Warning: #e5c07b, Error: #e06c75

Use it by setting in your ~/.config/yazi/theme.toml:

```toml
# Your overrides ...

theme = "onedark"
```

Yazi will merge your theme with this flavor automatically.
