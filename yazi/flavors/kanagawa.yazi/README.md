# Kanagawa Yazi Flavor

This directory provides the Kanagawa flavor for Yazi.

- Elegant, warm Japanese-inspired palette based on kanagawa-wave
- Background: #1F1F28, Foreground: #DCD7BA
- Primary accent: #7E9CD8, Secondary: #7FB4CA, Success: #98BB6C, Warning: #E6C384, Error: #C34043

Use it by setting in your ~/.config/yazi/theme.toml:

```toml
# Your overrides ...

theme = "kanagawa"
```

Yazi will merge your theme with this flavor automatically.
