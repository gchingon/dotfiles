# Tokyonight Yazi Flavor

This directory provides the Tokyonight flavor for Yazi.

- Dark, cool blue-themed variant based on tokyonight-night
- Background: #1a1b26, Foreground: #c0caf5
- Primary accent: #7aa2f7, Secondary: #7dcfff, Success: #9ece6a, Warning: #e0af68, Error: #f7768e

Use it by setting in your ~/.config/yazi/theme.toml:

```toml
# Your overrides ...

theme = "tokyonight"
```

Yazi will merge your theme with this flavor automatically.
